# Core Workflows

## Workflow 1: Tao LSX moi
1. User vao man Lenh san xuat.
2. User tao thong tin dau don.
3. He thong tao `ProductionOrderHeader`.
4. LSX xuat hien trong danh sach LSX.
5. Neu chua co giao dich NVL, LSX xuat hien trong khu cho cap nhat NVL.

## Workflow 2: Tu LSX sang NK NVL
1. User chon LSX chua co giao dich.
2. User bam "Bat dau cap nhat".
3. Form NK NVL mo voi du lieu prefill.
4. User nhap phat sinh thuc te.
5. He thong tao `MaterialMovement`.

## Workflow 3: Mo lai cap nhat NVL
1. User mo lai cung ma LSX.
2. He thong uu tien lay draft gan nhat.
3. Neu khong co draft, lay giao dich gan nhat.
4. Chi neu LSX chua co gi moi lay tu header goc.

## Workflow 4: Chot LSX
1. User kiem tra du giao dich.
2. User chot LSX.
3. He thong khoa them, xoa, doi trang thai giao dich lien quan.

## Workflow 5: Bao cao
1. Lay du lieu tu MaterialMovement.
2. Tong hop theo LSX, tho, cong doan, NVL.
3. Hien thi trong ton hop tho va bao cao hao hut.
