# Workflow Ton hop tho

1. He thong tong hop giao dich NVL theo ky bao cao.
2. Gom theo tho, cong doan, kim loai va tuoi vang.
3. Tinh ton dau ky, nhap, xuat, ton bot may, ton so sach, ton thuc te.
4. Danh dau trang thai soat xet va nhan xet.

## Rule
- UI uu tien dong can xu ly truoc.
- Chi hien thi nhom truong can quyet dinh tren man chinh.
