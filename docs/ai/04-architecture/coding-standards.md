# Coding Standards

## Naming
- Ten bien phai ro nghia nghiep vu.
- Khong dung ten mo ho nhu `data`, `item2`, `temp`.

## Function design
- Mot function chi lam mot viec.
- Function co side effect phai dat ten ro rang.
- Validation va transform phai co ten tach biet.

## UI component
- Component render khong nen chua qua nhieu rule tinh toan.
- Form field phai co label ro rang.
- Khong de table va business logic tron trong cung mot khoi qua dai neu tach duoc.

## Data safety
- Moi field dropdown phai lay tu source ro rang.
- Moi thay doi trang thai phai qua function co rule.
- Khong de fallback ngam gay ghi de du lieu user.

## Maintainability
- Neu mot task doi workflow, cap nhat docs truoc hoac ngay sau khi code.
- Neu co bug lien quan nghiep vu, viet them rule vao docs.
